# The Uncanny Valley

Магистерская диссертация МФТИ. Разработка функции потерь для обеспечения эмоциональной согласованности между речью и мимикой при дообучении аудио-управляемых моделей генерации говорящих лиц.

**Автор:** Почтар Катрин Викторовна, Центр «Пуск», МФТИ.
**Конференция:** 68-я Всероссийская научная конференция МФТИ, секция когнитивных технологий, 4 апреля 2026 г.

---

## Краткое содержание

Исследование проверяет, можно ли при дообучении уже развёрнутых архитектур генерации говорящих лиц (Wav2Lip, SadTalker) повысить эмоциональную согласованность между речью и мимикой через надстраиваемую кросс-модальную функцию потерь, не меняя саму архитектуру и не требуя эмоциональных меток на этапе генерации.

**Гипотезы:**
- **H1** (Wav2Lip): прирост macro F1 эмоциональной классификации не менее +10% при сохранении lip-sync.
- **H2** (SadTalker): тот же эффект на 3DMM-архитектуре.

**Ключевой результат:**
- H1: ΔF1 = +10,7 на тесте (0,611 → 0,718), бутстрап-95 %-CI на одном из внешних эвалюаторов не включает ноль.
- H2: значимый эффект на коэффициент-уровне (ΔF1 = +0,143, McNemar p = 0,0095); на rendered-уровне не подтверждён.

Полные результаты — в `main.tex` (текст ВКР) и в W&B-проектах (ссылки ниже).

## Пайплайн воспроизведения

Все ноутбуки запускать последовательно по нумерации. Каждый ноутбук опирается на артефакты предыдущих.

| № | Файл | Что делает | Главные выходы |
|---|---|---|---|
| 01 | `01_data_preprocessing.ipynb` | Готовит RAVDESS: разбивает на train/val/test по актёрам (без пересечений), извлекает кадры лица 96×96 для Wav2Lip и 224×224 для TimeSformer, строит мел-спектрограммы | `metadata.json` со сплитами 544/96/96 (4 эмоции) |
| 02 | `02_train_emotion_encoders.ipynb` | Дообучает 11 аудио- и 11 видео-конфигураций энкодеров эмоций; отбирает лучшие | Wav2Vec2-er (val F1 = 0,826), TimeSformer-8f (val F1 = 0,790) |
| 03 | `03_emotion_utils.ipynb` | Утилиты загрузки и инференса отобранных энкодеров (используется этапами 06–11) | Помощники Python для следующих этапов |
| 04 | `04_encoder_ceiling.ipynb` | Замеряет «потолок задачи» — F1 внутренних энкодеров на реальных тестовых кадрах | Аудио test F1 = 0,739; видео test F1 = 0,876 |
| 05 | `05_external_classifier_screening.ipynb` | Прогоняет 6 внешних FER-классификаторов на чистом RAVDESS, отбирает 2 для финальной оценки | Отобраны `motheecreator/vit-FER` и `Rajaram1996/FacialEmoRecog` |
| 06 | `06_wav2lip_loss_ablation.ipynb` | Сравнивает 5 конфигураций функции потерь на Wav2Lip: baseline, cos-only, ce-only, ce-cos, ce-kl | CE+KL побеждает на валидации (F1 = 0,737) |
| 07 | `07_wav2lip_finetune_H1.ipynb` | Файнтюнит Wav2Lip с CE+KL при разных масштабах s, проверяет H1 на test | Финальная модель `wav2lip-cekl-01`: F1 0,611 → 0,718, LSE-C изменился незначимо |
| 08 | `08_wav2lip_external_evaluation.ipynb` | Оценивает все Wav2Lip-конфигурации двумя внешними FER, считает McNemar и cross-classifier agreement | 8 из 8 эмоциональных конфигов проходят cross-classifier agreement |
| 09 | `09_sadtalker_loss_ablation.ipynb` | Та же 5-конфигурационная абляция, но для SadTalker | CE+COS побеждает (val F1 = 0,651) |
| 10 | `10_sadtalker_finetune_H2.ipynb` | Файнтюнит SadTalker с победителем абляции, проверяет H2 | На коэффициент-уровне ΔF1 = +0,143, p = 0,0095; на rendered эффект не подтверждён |
| 11 | `11_bootstrap_analysis.ipynb` | Бутстрап-95 %-CI на ΔF1 для всех пар «конфигурация × эвалюатор × сплит», per-actor sign-test | Для cekl-05 на Rajaram1996/test CI = [+0,004; +0,176] — значим |

## Симметрия методологии

Этапы 06–07 (Wav2Lip) и 09–10 (SadTalker) построены в одинаковой методологической рамке: одинаковая абляция, одинаковые внутренние энкодеры, одинаковый протокол внешней оценки. Различие — только в точке интеграции функции потерь, что и составляет содержание сравнения двух архитектур в разделе «Дискуссия» текста ВКР.

```
Подготовка                01 → 02 → 03 → 04
Скрининг эвалюаторов      05
Wav2Lip                   06 (абляция) → 07 (файнтюнинг + H1) → 08 (внешняя оценка)
SadTalker                 09 (абляция)                       → 10 (файнтюнинг + H2)
Статистика                11 (бутстрап + per-actor)
```

## Аппаратные требования и зависимости

- **GPU:** один NVIDIA T4 (16 GB) для большинства этапов; для SadTalker (этап 10) — A100 (40 GB) либо RTX 3090 (24 GB).
- **Python:** 3.10. PyTorch 2.1, transformers 4.40, librosa 0.10, scipy 1.11, wandb 0.16.
- **Воспроизводимость:** фиксированный seed = 42 для всех numpy/torch/cuda; `torch.backends.cudnn.deterministic = True`.
- Полный список зависимостей — в первой ячейке каждого ноутбука.

Большинство ноутбуков запускались в Google Colab к подключенному локально RTX 3090. Любая среда с CUDA 12.x и указанными версиями библиотек подходит.
